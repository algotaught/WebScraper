Flipkart Review Scraper

This webapp scraps flipkart for reviews.

Due to the limitations of the Heroku cloud deployment server this scraper fails to retrieve reviews if there are a lot of reviews for a product therefore I have limited the number of reviews so the server doesn't crash. But on local system/network when running the script locally it works perfectly fine.


The code and relevant files are available in the github repo, link available under about-me section.
